# CONSTRUCTION-CONTRACTS-AND-LEGAL-JUDGEMENTS
# 資料處理
## API串接（V1_request.py)
- 使用方式
1. 申請司法院開放平台的帳密，替換以下資訊
  - username=你的帳號
  - password=你的密碼
2. 程式運行
```shell
python V1_request.py
```
